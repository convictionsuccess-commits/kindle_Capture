# 🚀 Kindle知識強奪プロジェクト

この場所は、最強スペック（13900K & 4090）を駆使して、本から「知能」を抜き出すための拠点です。

## 🛠️ 使う道具

* **CPU**: Intel Core i9-13900K (制御塔)

* **GPU**: NVIDIA GeForce RTX 4090 (OCRエンジン)

* **メモリ**: 64GB (大量テキスト処理用)

## 📁 ファイルの説明

* **kindle_to_text_[4090.py*](http://4090.py)*: 4090でKindleを自動で読み取るメインプログラム。

## 🏃 実行する時の手順

1. **LM Studio** を起動して、モデル（Qwen2.5-VL）を読み込む。

2. **Wi-Fiを切る**（これ重要！）。

3. **Kindle** を1ページ目にして全画面で開く。

4. Cursorの画面下でプログラムを実行する。
